ADVANCED DEV TERMINAL

Welcome to the Advanced Dev Terminal!
Type help to see available commands.
>
Available Commands:
  • whoami: Show your info
  • projects: List portfolio projects
  • contact: Show contact info
  • run facebook: Open Facebook
  • run github: Open GitHub
  • run portfolio: Open Portfolio
  • back: Return to main Linktree
  • clear: Clear the terminal screen
  • help: Show this help box
  • echo <text>: Echo your text
  • date: Show current date and time
  • calc <expression>: Calculate math expression (e.g. calc 2+2*3)
